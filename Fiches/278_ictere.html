Item 278

Ictère de l'adulte et de l'enfant

Approche diagnostique et thérapeutique — adulte et nouveau-né
Hépato-gastroentérologie Pédiatrie Urgences
Vision globale. Cet item couvre deux terrains distincts : le nouveau-né (fréquent, physiologique dans 2/3 des cas mais potentiellement neurotoxique) et l'adulte (orientation selon le type de bilirubine et les étiologies). L'enjeu commun est de reconnaître les 3 urgences (ictère nucléaire, hémolyse grave, insuffisance hépatique sévère/angiocholite) et de dérouler une stratégie diagnostique structurée : biologique d'abord, imagerie ensuite.

3 urgences néonatales
1. Ictère nucléaire (= encéphalopathie hyperbilirubinémique) : bilirubine libre neurotoxique → léthargie, hypertonie extrapyramidale → séquelles : surdité, mouvements anormaux, lésions NGC
2. Hémolyse grave (incompatibilité ABO/Rhésus, déficit G6PD, sphérocytose héréditaire) → évolution rapide vers niveaux neurotoxiques + anémie
3. Atrésie des voies biliaires (bili conjuguée) → hépatomégalie ferme, urines foncées, selles décolorées → vit K + chirurgie urgente
Facteurs de risque d'ictère pathologique néonatal
Anamnestiques
  • AG < 38 SA
  • Incompatibilité ABO (mère O, enfant A ou B)
  • RAI maternelles +
  • Hémolyse familiale / fratrie traitée
  • Origine Asie, Afrique, Antilles
  • Survenue avant 24 h
  • Durée > 10 jours
Cliniques
  • Perte pondérale < 8 %
  • Bosse séro-sanguine, hématomes
  • Ictère jusqu'à la plante des pieds
  • Examen neurologique anormal
  • Anémie hémolytique (pâleur, HSM)
  • Cholestase (urines foncées, selles décolorées)
Stratégie diagnostique néonatale
Pas de bilan si ictère simple. Bilan si bili TC ≥ 75e percentile, facteurs de risque, ou ictère prolongé (> 7 j à terme, > 14 j prématuré)

Piège — Cholestase prolongée : carence en vitamines liposolubles A, D, E, K → TP bas avec facteur V normal (synthèse hépatique conservée) + stéatorrhée + ostéopénie.
A distinguer de l'IHC : TP bas + facteur V bas (défaut de synthèse hépatique).
Insuffisance hépatocellulaire grave — Signes et complications
Critères de gravité
  • TP + facteur V < 50 %
  • Encéphalopathie hépatique
  • Ictère intense, souvent mixte
  • Scores : Child-Pugh, MELD
Complications
  • Hémo : vasodilatation, hypoTA, tachycardie
  • Neuro : coma, œdème cérébral (aiguë seulement)
  • Respi : hypoxie, hypocapnie, SDRA
  • Rénal : IRA fonctionnelle, syndrome hépatorénal
  • Métab : hypoglycémie, hyponatrémie, acidose
  • Infect : pneumopathie, septicémie (bact. + fungi)
Mortalité spontanée de l'IHC grave avec encéphalopathie : ~80 % → soins intensifs spécialisés + rapprochement centre de transplantation hépatique

Échographie dans l'ictère conjugué : non invasive, peu coûteuse, sans danger. Objectif principal = chercher une dilatation des voies biliaires (obstruction VBP / convergence hépatique). Sensibilité/spécificité variables selon opérateur et habitus. Complétée par imagerie de 2e ligne si nécessaire.
Urgences devant un ictère de l'adulte
1. Angiocholite (cf. item 277) : triade de Charcot = ictère + fièvre + douleur biliaire
2. Ictère du cirrhotique : chercher facteur aggravant (hépatite virale, alcoolique, auto-immune, CHC, infection bactérienne)
3. Insuffisance hépatique aiguë : transaminases > 20×N + TP/FV < 50 % ± encéphalopathie → mortalité 80 % → transfert centre de transplantation
Points-clés à retenir
Pièges classiques ECNi
  • Ictère avant 24 h de vie → toujours pathologique, jamais simple
  • Selles décolorées chez un nourrisson → atrésie biliaire jusqu'à preuve du contraire → urgence chirurgicale
  • Gilbert = bilirubine NC < 80, bilan hépatique normal, pas d'hémolyse → pas de suivi
  • TP bas + FV bas → IHC. TP bas + FV normal → cholestase (carence vit K)
  • Grosse vésicule indolore + ictère cholestatique = signe de Courvoisier → cancer sous-ampullaire
  • Bili mixte + transaminases > 20×N → hépatite aiguë grave → urgence transplantation
Réflexes diagnostiques
  • 1er geste = bili totale + conjuguée → oriente tout l'arbre décisionnel
  • Bili conjuguée → échographie obligatoire → dilatée ou non dilatée ?
  • Anti-mitochondries → CBP (prurit dominant, femme 40–60 ans)
  • CholangoIRM → CSP (MICI associée souvent)
  • Ictère + anémie + réticulocytes ↑ = hémolyse (NC)
  • Photothérapie = traitement de base de l'ictère libre néonatal
